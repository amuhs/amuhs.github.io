Description

A brief, informal, introduction to web development; this workshop was designed to provide an overview of html and css in order to provide a start to becoming a web developer.

Students begin with a skeleton html file and are instructed in its different parts. Once this is completed students are shown a basic web page and asked to duplicate it. Resources as well as a few nudges in the right direction are also provided.

Once the challenge is complete, the source code is shown, explained further, and the process starts over again.

There are a total of three challenges, each more difficult than the last and developed to teach the basic fundamentals of html and css.


What You Need

  • Laptop
  • Text Editor
  • Internet Connection*

While suing a text editor is not required, it is strongly recommended. Using Notepad, in Windows, or TextEdit, on a Mac, can cause headaches when trying to use file extensions other than ".txt".

My recommendation is Atom by GitHub. It is easy to use and allows you to see your entire project directory while you are working.

*An internet connection is not required if you do not link to a website or a file on a web server.


Setup

There are two ways to prepare for your workshop. You can download the web-workshop.zip file, and call it a day. Or you can follow the steps below.

  1. Create a folder to contain your project. I suggest, "web-workshop".
  2. Inside of this folder, create a file called, "index.html".
    1. This can be down with a simple word processor I suggest using a text editor designed for this task. See "What You Need" above.
    2. As well as being the homepage for your web project, this will be the main file used in this workshop.
  3. While still inside the project, or parent, folder, create a folder named, "css".
  4. Move inside the newly created "css" folder.
    1. You should now be two folders deep: ../web-workshop/css
  5. Create a file called, "main.css".
